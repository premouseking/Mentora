# 学习计划重设计 v11：阶段-章节-任务标准划分

> 版本：v11 | 日期：2026-07-01 | 状态：设计中

| 阶段 | 阶段简介 | 章节划分 | 任务分配 | 任务类型 |
|------|---------|---------|---------|---------|
| **基础知识** | 覆盖课程全部章节，建立完整知识框架 | 按照课程本身的进度章节划分，比如第一章 计算机系统概述 | 知识点 + 练习。知识点列举章节中的重要知识点，练习是对应的练习，每个章节至少一个练习 | 知识点 + 练习 |
| **重点突破** | 聚焦核心难点知识块，深入理解重点内容 | 按照重要的知识块划分，每个章节是一个完整的知识体系 | 知识点 + 练习。按照知识体系中每个细分知识点划分，每个章节至少一个练习 | 知识点 + 练习 |
| **专项训练** | 按题型分类练习，提升解题技巧和速度 | 按照题型划分，一个题型一个章节 | 全部为练习。每个章节的任务有基础练习、进阶练习 | 练习 |
| **综合应用** | 练习综合大题，培养跨章节知识整合能力 | 专门练习综合大题，按照大题题型划分，一个章节一个大题 | 讲解 + 练习。先讲解解题思路，再做练习 | 讲解 + 练习 |
| **真题练习** | 模拟真实考试环境，检验整体复习效果 | 按照真题套题划分，一套题一章 | 全部为练习。按照题型依次划分：选择、填空、简答等 | 练习 |
| **易错题** | 针对常见易错考点，避免重复犯错 | 按照大的考点划分 | 全部为练习。每个任务对应该考点的一个小的易错点 | 练习 |
| **复习巩固** | 再次梳理重点章节，强化关键知识点记忆 | 按照课程本身的进度章节划分 | 知识点 + 练习。只包含部分重点知识点，每个章节至少一个练习 | 知识点 + 练习 |
| **错题再练** | 重做历史错题，确保薄弱环节不再出错 | 按照课程本身的进度章节划分 | 全部为练习。对应章节里做错过的题 | 练习 |
