│  导  │    [Task]─┐         ┌─[Task]          │           │

### 4.2 左栏 — 纵向阶段导航

与建课确认页阶段导航逻辑一致，方向从横变纵：

- **所有阶段**纵向排列在一条可滚动滑轨上
- **至多可见 4 个**阶段块（`overflow-y: auto` + `scroll-snap-type: y mandatory`）
- **滚轮驱动**：滚轮控制阶段切换，选中块同步更新
- **点击切换**：点击某个阶段块，中栏画板切换到对应阶段的主干分支图
- **选中态**：当前阶段高亮，其余灰态
- 阶段数 ≤ 4 时无需视口位移

```
┌──────────┐
│  阶段 1  │  ← 选中
├──────────┤
│  阶段 2  │
├──────────┤
│  阶段 3  │
├──────────┤
│  阶段 4  │
│          │  ← 滚动可见更多
│  阶段 5  │
└──────────┘
```

### 4.3 中栏 — 主干 + 分支画板

#### 整体结构

- 画板容器：`overflow: auto`，内容超出时可上下左右滚动
- 当前选中阶段的所有章节（Chapter）组成**主干**，从上到下依次排列
- 章节之间用**竖线**连接
- 每个章节向左右交替展开**分支**，连接到该章节下的任务（Task）选项卡

#### 主干：章节节点

- 章节为**矩形选项卡**，居中排列在画板中轴线上
- 章节之间用竖线连接，形成视觉上的主干
- 选中某个章节可高亮该章节及其所有子任务

#### 分支：任务选项卡

- 任务为较小的**矩形选项卡**
- 交替规则：第奇数个章节的任务在右边，第偶数个在左边
  - Chapter 0 → 任务在右侧
  - Chapter 1 → 任务在左侧
  - Chapter 2 → 任务在右侧
  - ……
- 同一章节的多个任务纵向排列在同一侧
- 分支线：从章节节点引出水平线，再分叉到各个任务

#### 连接线绘制方式

使用 SVG `<path>` 或 `<line>` 绘制连线：
- **主干竖线**：章节节点之间的垂直连线
- **分支横线**：从章节节点到任务节点的水平连线
- **任务分叉**：横线末端分叉到各个任务

#### 防碰撞

- 每个章节的 Y 坐标预留足够空间：`max(章节块高, 任务数 × 任务间距)`
- 相邻章节间隔固定 `gap`，避免上下分支重叠

#### 交互

- 画板整体可拖拽/滚轮滚动（`overflow: auto`）
- 点击任务选项卡 → 右栏展示该任务详情
- 点击章节选项卡 → 右栏展示该章节概览
- 选中态：高亮当前选中节点

### 4.4 右栏 — 可收起详情栏

#### 默认状态

展开，宽度固定（如 280px），内容留空占位。

#### 收起/展开按钮

- 一个小按钮（如 `◀` / `▶`），位于右栏与中栏交界处或右栏顶部
- 点击收起：右栏宽度收为 0 或极窄（如 20px 显示图标），中栏扩展
- 点击展开：恢复默认布局
- 过渡动画：`transition: width 200ms`

#### 内容（Mock 占位）

当前阶段详情留空，Mock 数据展示：
- 选中 Task 时：任务名称、任务类型、预估时长、关联资料（留空）
- 选中 Chapter 时：章节说明、知识点概览
- 以上均为占位文本，不承载真实数据

### 4.5 顶部栏 — 关闭机制

因中栏画板需要鼠标拖拽滚动，与原来的下拉关闭手势冲突，v6 修改为：

- **移除**：`onPointerDown/Move/Up` 下拉关闭逻辑
- **保留**：顶部"下拉收起"提示文字旁的 **X 关闭按钮**（`quiz-hint-close`）
- 提示文字从"下拉收起"改为"学习方案概览"或类似静态文案
- X 按钮始终可见、始终可用

## 五、与 v5 的差异总结

| 区域 | v5（沿用 v4） | v6 |
|------|-------------|-----|
| 左栏 | 无独立栏，阶段在列表中展示 | 纵向阶段导航栏，滚轮 + 点击切换，至多 4 个 |
| 中栏 | 纵向列表（阶段 → 单元 → 任务） | 主干 + 分支画板，章节为主干，任务左右交替分支 |
| 右栏 | 无 | 可收起详情栏，当前留空 |
| 关闭方式 | 下拉手势 + X 按钮 | **仅 X 按钮**（移除下拉手势，避免与画板拖拽冲突） |
| 方案概览 | 覆盖层内展示 | **移除**（阶段级别信息移至左栏，任务级别信息移至右栏） |
| 底部按钮 | 调整计划 + 开始学习 | 仅调整计划（开始学习入口已在页面外部） |
