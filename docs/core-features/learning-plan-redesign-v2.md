### 阶段导航栏（上方）

**交互**：
- 默认横向滚动展示所有阶段（4-5 个）
- 鼠标悬停可预览阶段名称
- 点击选中，下方内容区切换
- 支持左右箭头滚动（阶段多时）

**视觉**：
- 箭头形卡片，当前选中高亮
- 阶段间用箭头连接，暗示顺序
- 已完成阶段显示进度标记

### 内容展示区（下方）

**结构**：
1. **阶段标题 + 目标描述**
2. **章节列表**（每个 Chapter 一张卡片）
   - 章节名称
   - 一句话简介
3. **阶段级调整按钮**：保持 / 精简 / 加强

**不展示**：
- Task 级别的知识点（太细）
- 学习档案详情（移到最下面）

### 学习档案（底部）

- 折叠状态，默认收起
- 点击展开，内容和前几步底栏一致
- 只显示关键信息摘要

## 二、阶段类型池（AI 建议 + 用户调整）

### 8 种标准类型

| 类型 | 适用场景 | 典型章节数 |
|------|---------|-----------|
| 基础入门 | 零基础起步 | 5-8 |
| 知识梳理 | 系统学习教材 | 按教材章节 |
| 专项训练 | 突破薄弱点 | 3-5（按题型/考点）|
| 综合应用 | 跨知识点实战 | 2-4（按综合卷/专题）|
| 实战演练 | 限时模拟 | 3-5（按套卷）|
| 查漏补缺 | 错题/薄弱强化 | 按错题类别 |
| 考前冲刺 | 考前最后阶段 | 2-3（高频考点+策略）|
| 拓展提升 | 学有余力延伸 | 按拓展内容 |

### AI 初版建议

根据学习档案自动选择 4-5 个阶段，排出顺序。例如：

| 档案特征 | AI 建议阶段序列 |
|---------|----------------|
| 高考冲刺、有基础、有截止 | 知识梳理 → 专项训练 → 综合应用 → 考前冲刺 |
| 零基础、无截止、兴趣驱动 | 基础入门 → 知识梳理 → 综合应用 → 拓展提升 |

### 用户调整

- **增删**：从 8 种类型池中添加或删除
- **调序**：拖拽改变阶段顺序
- **改名**：允许自定义阶段名称
- **调整完点击「重新生成」**，AI 根据新阶段列表重新规划 Chapter

## 三、阶段级调整（替代原有的简化/加强按钮）

每个阶段卡片旁提供三个选项：

| 选项 | 效果 |
|------|------|
| **保持** | 不变 |
| **精简** | 减少 1-2 个 Chapter，每个 Chapter 减少一句话描述深度 |
| **加强** | 增加 1 个 Chapter（拆细），或增加深度描述 |

调整后点击「重新生成」，AI 仅调整该阶段内容，其他阶段保持不变。

## 四、课程页上划栏（/courses/{id}）

### 与建课确认页的差异

| | 建课确认页 | 课程页上划栏 |
|---|---|---|
| **目标** | 审视方案、确认调整 | 查看进度、继续学习 |
| **阶段导航** | 可点击切换查看 | 当前阶段高亮，可切换 |
| **章节展示** | 全部章节（预览） | 仅当前阶段章节 + 完成状态 |
| **操作按钮** | 保持/精简/加强 + 开始学习 | 继续学习 |

### 上划栏布局

```

│  基础梳理  ──▶  【专项突破】──▶  综合应用      │
│       ✅              ◐                  ○    │  ← 进度标记
```
