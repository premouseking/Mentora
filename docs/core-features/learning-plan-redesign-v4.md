## 二、调整计划的设计原则

- **入口独立**：不直接修改展示面板，点「调整计划」按钮进入专用调整界面
- **两页各有独立的、完整的调整 UI**：工作量较大，后续单独设计
- **安全**：面板只读，调整操作在专用界面完成，确认后统一写回

## 五、阶段类型池（8 种）与 AI 分阶段 prompt 策略

| 类型 | 适用场景 |
|------|---------|
| 基础入门 | 零基础起步，核心概念和基本方法 |
| 知识梳理 | 系统学习教材，按章节深入 |
| 专项训练 | 突破薄弱点，题型分类训练 |
| 综合应用 | 跨知识点实战，真题/项目 |
| 实战演练 | 限时模拟，成套练习 |
| 查漏补缺 | 错题回看，高频错误强化 |
| 考前冲刺 | 考点回顾，临场策略 |
| 拓展提升 | 课外延伸，竞赛/前沿方向 |

### AI 分阶段策略

Prompt 需引导 AI 根据不同学习档案调整阶段分配：

| 档案特征 | 建议阶段序列 | 权重倾向 |
|----------|-------------|---------|
| 零基础冲刺考试 | 基础入门 → 专项训练 → 考前冲刺 | 60/25/15 |
| 有基础冲刺考试 | 知识梳理 → 专项训练 → 综合应用 → 考前冲刺 | 20/30/30/20 |
| 系统学习无截止 | 基础入门 → 知识梳理 → 综合应用 → 拓展提升 | 20/40/25/15 |
| 快速入门 | 基础入门 → 专项训练 → 实战演练 | 50/30/20 |
| 考前突击 | 查漏补缺 → 考前冲刺 | 40/60 |
| 补弱专项 | 专项训练 → 综合应用 | 70/30 |

**prompt 策略规则**：
1. **先看截止日期**：紧迫 → 压缩基础，加大冲刺/查漏
2. **再看当前基础**：零基础 → 基础入门权重高，不得跳过
3. **再看学习目标**：考试类 → 必含考前冲刺；兴趣类 → 可含拓展提升
4. **最后看时间预算**：时间少 → 减少阶段数（3-4 个）；时间充足 → 5 个
